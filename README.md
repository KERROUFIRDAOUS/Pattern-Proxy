# Pattern-Proxy

## Pattern Proxy
Le Pattern Proxy fournit un intérmédiaire entre la partie cliente et un bojet pour controler les accès à ce dernier. Le Pattern Proxy permet d'isoler le comportmenet lors de l'accès à un objet.
